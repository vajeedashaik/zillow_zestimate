- Existing site: Next.js 16 with Tailwind CSS, Recharts, dark theme

TASK:
Build an interactive geographic heatmap dashboard as a Next.js page/component that visualises Zestimate error hotspots.

8. Match the existing dark theme (Tailwind dark classes, same colour palette as current site)

OUTPUT:
